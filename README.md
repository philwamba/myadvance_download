# MyAdvance Android App

The latest debug build of the MyAdvance Android application is available here.

## Download Links

- [Download APK](downloads/myadvance-debug.apk)

## Installation Instructions

1.  Download the `myadvance-debug.apk` file to your Android device.
2.  Enable "Install from Unknown Sources" in your device settings if prompted.
3.  Tap the downloaded file to install.
