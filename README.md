## Installation Instructions

1. Download the My Advance APK file to your Android device.
2. Enable "Install from Unknown Sources" in your device settings if prompted.
3. Tap the downloaded file to install.
4. **Note for Release build**: Since this is an unsigned build, some devices may require additional confirmation or might not allow installation depending on security settings.
